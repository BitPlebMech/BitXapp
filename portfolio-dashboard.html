GitHub Gist Sync
First-time setup:
1. Go to github.com/settings/tokens/new
2. Name it "Portfolio" · tick gist scope only · Generate
3. Paste token below → Save Settings → click Save to Gist
4. Copy the Gist ID that appears and save it with your token in Notes

On any other browser or after clearing cache:
Paste your saved token + Gist ID → Save Settings → Load from Gist.
Both fields are needed — treat the Gist ID like a second password.

Use the Gist button in the header to sync manually.
